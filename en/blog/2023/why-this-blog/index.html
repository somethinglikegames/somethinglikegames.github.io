The reason for this blog

January 1, 2023
Miscellaneous

Welcome to our site!

Nice that you take the trouble to read this text. Here I would like to explain my intention first, before I start with the realization, therefore now this post.

Secretly, I’ve been involved in game development since my childhood. More precisely, since the early 1990s, when I was allowed to play games like Pirates! or California Games on my brother’s C64. Game development was also the original reason I became a software developer in the first place. As a teenager, I always imagined myself working at id Software someday with legends like John Carmack. Regardless of the fact that this wouldn’t be possible in the current time as the world doesn’t stop for my sake, I never made the professional entry into game development. Since I earn money with software development, I do this by developing enterprise software in the broadest sense. But some time ago I started again to get more concretely involved with game development, and I see the great advantage of being able to incorporate my knowledge from other fields. And so the idea for this blog was born.

In order to do justice to all those who prefer to read in German rather than English, I would like to make this site bilingual.
